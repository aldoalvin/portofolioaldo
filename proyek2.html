Detail Proyek: Puzzle Game Coding

Logic Paradox adalah sebuah proyek pembelajaran coding menggunakan bahasa Python. Proyek ini terdiri dari 10 level, dimana setiap level merupakan tantangan yang dirancang untuk meningkatkan kemampuan pemrograman secara bertahap. Game ini menggunakan HTML sebagai strukturnya, CSS untuk desainnya dan Javascript yang membuat lebih dinamis.

Catatan: Logic Paradox merupakan karya saya di ekstrakurikuler Informatika Lanjut.

Fitur Utama:

  • Pengajaran bahasa pemrograman Python secara bertahap
  • 10 level tantangan yang meningkatkan kemampuan coding
  • Tantangan yang dirancang untuk mengasah logika dan pemecahan masalah
  • Antarmuka yang user-friendly dan mudah digunakan
  • Materi pembelajaran yang interaktif dan menyenangkan
